Button 
Connect a push button and use it as an input to turn an LED    on and off.

In this example, we show how to detect when the user presses a push button. When the button is not pressed, there is no connection between pin 2 and GND, and the result is that the value of pin 2 will be 5V (HIGH). When, on the other hand, someone presses the button, the value of the pin becomes 0V (LOW). To really understand the operation of this circuit, you need some basic knowledge of electronics, and so we won't go into it here. The main thing is that depending on whether the button is pressed or not, the value of pin 2 becomes 0V or 5V respectively. Note that unlike in the LED example, where the Arduino makes the pin HIGH or LOW, in our button example here, it is the external circuitry that makes the pin HIGH or LOW. The role of the Arduino is now to detect or read the value of the pin voltage (in the LED example, the Arduino set or wrote the value of the pin voltage).

In the code, the pinMode() for pin 2 is set to INPUT, meaning that we can now read from this pin. To do this, we use the function digitalRead(), which returns HIGH or LOW, depending on whether the voltage of the pin is closer to 5V or 0V.

Note that the code has an LED is connected to pin 9. We can use the schematic from LED Blink to add the LED. Alternatively, we could also change the code to use an LED at pin 13. This particular pin (pin 13) has an LED on the board itself that we can turn on this way. So while we can add the external LED as in the Blink example, we can also just omit it and observe the on-board LED turning on and off when we press the button (provided we change the code to use pin 13 for the LED).

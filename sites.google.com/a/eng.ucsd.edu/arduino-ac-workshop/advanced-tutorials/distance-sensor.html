Distance sensor


 
 
 

 
Ultrasound     Distance Sensor 

This sensor allows you to measure the distance to a object.


The ultrasound sensor lets you measure distances, much like a bat does or how sonar works. It sends out a sound signal and then listens for the reflection of that sound on far-away objects. If you calculate how long it takes for this reflection to reach you, and you know the speed of sound, you can calculate how far off  the object is. And much like for a bat, the sound signals used by this sensor are in the ultrasound range, meaning that they are sounds that are outside of our human hearing.

The diagram below shows how to hook up the sensor. Please look at the names for the pins on the sensor to make sure you connect everything correctly.

The code was taken from this tutorial http://howtomechatronics.com/tutorials/arduino/ultrasonic-sensor-hc-sr04/. It also explains quite nicely how the sensor works, so if you have the time, please have a look at it.
   


  
/***********************************************************************************************
    Tutorial Ultrasound Distance Sensor

    The ultrasound distance sensor sends a sound signal and listens for the reflection. Based on
    the speed of sound, the distance can be measured.
    To view the result, start the Serial Monitor.

    Code from https://howtomechatronics.com/tutorials/arduino/ultrasonic-sensor-hc-sr04/
    
*************************************************************************************************/


// Pins to use 
const int trigPin = 3;
const int echoPin = 4;

// Declare the variables
long duration;
int distance;


void setup() {
  pinMode(trigPin, OUTPUT);           // Sets the trigPin as an Output
  pinMode(echoPin, INPUT);            // Sets the echoPin as an Input
  Serial.begin(9600);                 // Starts the serial communication
}


void loop() {
  
  // Clear the trigPin
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);

  // Set the trigPin to HIGH for 10 micro seconds
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);

  // Read the echoPin and return the sound wave travel time in microseconds
  duration = pulseIn(echoPin, HIGH);

  // Calculate the distance in cm
  distance= duration*0.034/2;

  // Print the distance on the Serial Monitor
  Serial.print("Distance: ");
  Serial.println(distance);

}

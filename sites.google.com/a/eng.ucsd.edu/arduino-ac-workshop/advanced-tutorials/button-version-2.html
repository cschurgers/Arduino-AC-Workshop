Button with Pullup 

A simplified way to use a push button. It uses an internal    pullup resistor.

In this example, we show how to detect when the user presses a push button. However, compared to the original Button tutorial, there is no resistor in the schematic. The reason is that the external resistor in now replaced by one that is built into the Arduino itself, called a pullup resistor. In the code, you will see that you can activate this pullup resistor, which is connected between the pin and 5V, by choosing the pinMode to be "INPUT_PULLUP". This means that the system is pretty much equivalent to that of the Button tutorial, but without the need to add that extra resistor yourself on the breadboard. For us, the most useful application of this feature is exactly what we do here: simplify our circuit for a push button.

Note that the code talks about an LED connected to pin 13. You can add the LED by merging the schematic from the LED Blink tutorial.  However, pin 13 (no other pin) also has an LED on the Arduino itself that we can turn on this way. So even if you don't connect an external LED, you should still see that internal LED turn on and off with the example code below.

 For more info on INPUT_PULLUP, you can check out https://www.arduino.cc/en/Reference/pinMode.
   


  
/**********************************************************************************
   Push Button Tutorial 2

   This tutorial demonstrates how to read a push button. The pin does NOT need to be 
   connected to an external resistor for this to work (see schematic). It uses an
   internal pullup resistor instead.
   It turns on the internal LED (associated with pin 13).
***********************************************************************************/

const int ledPin = 13;                    // LED pin
const int buttonPin = 2;                  // Push button pin 
int buttonState;                          // Variable to store state

void setup() {
  pinMode(ledPin, OUTPUT);                // The digital pin configured as an output
  pinMode(buttonPin, INPUT_PULLUP);       // The digital pin configured as an input with a pullup
                                          // i.e., there is an internal pullup resistor
                                          // If the pin is not connected to anything, it will read HIGH
}

void loop () {
 
  buttonState = digitalRead(buttonPin);   // Read from button pin (HIGH or LOW)
 
  if (buttonState == LOW)                 // Note: if the button is not pressed, the button pin is HIGH 
                                          // because of internal pullup 
   digitalWrite(ledPin, HIGH);            // Turn on LED
  else
    digitalWrite(ledPin, LOW);            // Turn off LED
}

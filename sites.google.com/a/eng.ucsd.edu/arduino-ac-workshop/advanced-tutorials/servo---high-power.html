Servo - high power


 
 
 

 
Servo - high power
Similar to the servo tutorial, but using an external battery pack  to provide more power. This is optional for micro servos, but more important for larger servos.

This tutorial is a continuation of the Servo tutorial. It is recommended to go through this tutorial first. In fact, the code is exactly the same. The only difference is that an external battery pack is now used to power the servo.

This setup is needed when the servo has to push hard or when we are using multiple servos. The reason is that the Arduino can only deliver limited current and the servo(s) may need more (you will find that servos start acting 'weird' if they don't get enough current). In this case, we can hook up external batteries (please note the connections). You can use battery packs with 3 or 4 AA batteries, providing either 3 x 1.5V = 4.5V or 4 x 1.5V = 6V.

As a side note, if we want to power the Arduino itself from with an external battery using the power plug (instead of the USB cable), we need a battery with a voltage between 7V and 12V (e.g., a 9V battery).

The code is identical to that of the servo tutorial. Also, as before, the servo control pin (in our example, pin 5) needs to be capable of PWM, or, in other words, it can only be connected to pins that have the '~' symbol.


   

  
/**********************************************************************************
   Servo Tutorial

   This tutorial demonstrates how to control a standard servo, i.e., a servo that
   can move a half circle (from 0 degrees to 180 degrees and back). 
   It uses a library called <Servo.h>
***********************************************************************************/

#include <Servo.h>                        // Include the <Servo.h> libary, which is specifically designed
                                          // to control servos
 
int servoPin = 5;                         // Servo pin number
Servo servo;                              // Define a special kind of abstract object of type "Servo"
                                          // This allows us to control the servo using the functions in the
                                          // <Servo.h> library

// Setup function
void setup() {                                 
  servo.attach(servoPin);                 // Attach the servo to the servoPin
}

// Main function (executed continuously)
void loop() {                                  
  
  servo.write(0);                         // Turn the servo to position 0 degrees
                                          // The number passed to the servo.write() function is the target servo position
  delay(2000);                            // Wait for 2000 ms
                                          // This delay is needed since it takes time for the servo to move to its new position.
                                          // The servo.write() command tells the servo where it needs to move to, but the program
                                          // immediately start executing the next lines of code, while the servo is moving.
                                          // In this case, we want to make sure the servo has enough time to move, before we go on 
                                          // with the rest of the program.
  servo.write(180);                       // Turn the servo to position 180 degrees
  delay(2000);

}

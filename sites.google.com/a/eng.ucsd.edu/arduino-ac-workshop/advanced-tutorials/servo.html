Servo 
Hooking up a standard servo that moves between the 0 and  180 degree angles.

This tutorial makes use of a dedicated library called Servo.h. A library is nothing more than a collection of useful functions that someone else wrote for us. We need to have a line at the start of your program "#include <Servo.h>" that tells the Arduino that it needs to use that Servo library and that we will be using some of its functions.

In the wiring diagram, note the colors of the wires. The red and black wires need to be connected to 5V and GND respectively. The third wire is the control wire, which is connected to the pin we write to in our program. This third wire can have a different color depending on the servo manufacturer (often, it is white or yellow). Note that the control pin (in our example, pin 5) needs to be capable of PWM (see also the explanation in LED Fade tutorial). This means that the servo only works with pins that have the '~' symbol. The reason is that the PWM signal is used to indicate the position of the servo (and the servo library takes care of the translation between the angle you want and the appropriate PWM control signal needed).

If the servo needs to 'work hard' or when you have multiple servos, it is possible the Arduino cannot deliver all the power it needs. In this case, you may notice the servo will start acting 'weird'. To solve this problem, you have to hook up the servo to an external battery pack, as explained in this Servo - high power tutorial.

As a side note, if we want to power the Arduino itself from with an external battery using the power plug (instead of the USB cable), we need a battery with a voltage between 7V and 12V (e.g., a 9V battery).

For more info on the Servo library, you can check out https://www.arduino.cc/en/reference/servo.

   

  
/**********************************************************************************
   Servo Tutorial

   This tutorial demonstrates how to control a standard servo, i.e., a servo that
   can move a half circle (from 0 degrees to 180 degrees and back). 
   It uses a library called <Servo.h>
***********************************************************************************/

#include <Servo.h>                        // Include the <Servo.h> libary, which is specifically designed
                                          // to control servos
 
int servoPin = 5;                         // Servo pin number
Servo servo;                              // Define a special kind of abstract object of type "Servo"
                                          // This allows us to control the servo using the functions in the
                                          // <Servo.h> library

// Setup function
void setup() {                                 
  servo.attach(servoPin);                 // Attach the servo to the servoPin
}

// Main function (executed continuously)
void loop() {                                  
  
  servo.write(0);                         // Turn the servo to position 0 degrees
                                          // The number passed to the servo.write() function is the target servo position
  delay(2000);                            // Wait for 2000 ms
                                          // This delay is needed since it takes time for the servo to move to its new position.
                                          // The servo.write() command tells the servo where it needs to move to, but the program
                                          // immediately start executing the next lines of code, while the servo is moving.
                                          // In this case, we want to make sure the servo has enough time to move, before we go on 
                                          // with the rest of the program.
  servo.write(180);                       // Turn the servo to position 180 degrees
  delay(2000);

}
